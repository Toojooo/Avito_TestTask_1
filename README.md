=== REST-приложение для управления Pull Request’ами, ревьюерами и командами. ===
Приложение реализует:
-создание PR
-назначение ревьюеров
-переназначение ревьюеров
-выборку PR по ревьюеру
-merge PR
-сбор простой статистики
-удобный Swagger-интерфейс

=== Стек технологий ===
-Java 21
-Spring Boot 
-Spring Web
-Spring Data JPA
-Hibernate
-PostgreSQL
-Lombok
-Swagger / OpenAPI 3
-JUnit 5 + Mockito

=== ЗАПУСК ===

Убедитесь, что установлены Docker и Docker Compose

Запустите приложение: docker-compose up -d

Приложение доступно по адресу: http://localhost:8080

Остановка: docker-compose down

=== API ENDPOINTS ===

-PRController — управление PR
Базовый путь: /api/prs


POST /api/prs — Создать PR
Создаёт новый PR от автора и автоматически назначает двух случайных активных ревьюеров из его команды.
Вход: PrCreateDto → Выход: PrResponseDto.

POST /api/prs/{id}/reassign — Переназначить ревьюера
Заменяет указанного ревьюера на нового кандидата из той же команды (случайный выбор).
Работает только для не-merge’d PR.

GET /api/prs/reviewer/{userId} — PR пользователя
Возвращает список PR, где пользователь назначен ревьюером.

PATCH /api/prs/{id}/merge — Смержить PR
Переводит PR в статус MERGED.
Недоступно для уже merge’d PR.


-StatsController — статистика назначений
Базовый путь: /api/stats

GET /api/stats — Получить статистику назначений
Возвращает агрегированную статистику:
• сколько раз каждый пользователь назначался ревьюером
• количество ревьюеров для каждого PR


-TeamController — управление командами
Базовый путь: /api/teams

POST /api/teams — Создать команду
Создаёт новую команду с указанным именем.
Вход: TeamDto (name).
Выход: TeamDto с присвоенным ID.


-UserController — управление пользователями
Базовый путь: /api/users

POST /api/users — Создать пользователя
Создаёт нового пользователя.
При необходимости можно сразу указать команду, к которой он принадлежит.
Вход: UserDto (имя, teamId).
Выход: созданный UserDto.

PATCH /api/users/{id}/active?active=... — Изменить статус пользователя
Меняет статус «активен / не активен» для указанного пользователя.
Возвращает 200 OK без тела.
